Class SenderReportPacket

import { SenderReportPacket } from 'rtp.js';

Representation of a RTCP Sender Report packet. It may contain various ReceiverReport instances.

serialize

  • serialize(): void

  • Overrides RtcpPacket.serialize

    • Defined in rtcp/senderReport.ts:268

    Apply pending changes into the packet and serialize it into a new internal buffer (the one that getBuffer will later return).

    NOTE: In most cases there is no need to use this method. It must be called only if the application retrieves information from the packet (by calling getBuffer, getReports, etc) and modifies the obtained buffers in place. However, it's recommended to use the existing setter methods instead (addReport, etc).

    Returns void
